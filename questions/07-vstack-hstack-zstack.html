What is the difference between VStack, HStack, and ZStack?

Answer in English

VStack (Vertical Stack):

Arranges views vertically from top to bottom.

VStack(alignment: .leading, spacing: 10) {
    Text("First")
    Text("Second")
    Text("Third")
}
┌─────────┐
│ First │
├─────────┤
│ Second │
├─────────┤
│ Third │
└─────────┘

HStack (Horizontal Stack):

Arranges views horizontally from left to right.

HStack(alignment: .top, spacing: 15) {
    Text("Left")
    Text("Center")
    Text("Right")
}
┌──────┬────────┬───────┐
│ Left │ Center │ Right │
└──────┴────────┴───────┘

ZStack (Depth Stack):

Layers views on top of each other along the Z-axis (depth).

ZStack(alignment: .topLeading) {
    Rectangle()
        .fill(Color.blue)
    Text("Overlay")
        .foregroundColor(.white)
}
┌─────────────┐
│ Overlay │ ← Text on top
│ │
│ Blue Rect │ ← Rectangle behind
│ │
└─────────────┘

Key Parameters:

Stack Alignment Options Direction
VStack .leading, .center, .trailing Top → Bottom
HStack .top, .center, .bottom Left → Right
ZStack .topLeading, .center, .bottomTrailing, etc. Back → Front

Common Use Cases:

  • VStack: Forms, lists, vertical menus
  • HStack: Navigation bars, button groups, horizontal layouts
  • ZStack: Overlays, backgrounds, layered designs

الشرح بالعربي

VStack (الترتيب العمودي):

بيرتب الـ Views من فوق لتحت، زي ما تكون بترص كتب فوق بعض.

HStack (الترتيب الأفقي):

بيرتب الـ Views من الشمال لليمين، زي ما تكون بترص أشياء جنب بعض على رف.

ZStack (الترتيب بالعمق):

بيحط الـ Views فوق بعض، زي ما تكون بتلزق استيكرز فوق بعض.

المعاملات المهمة:

  • alignment: بيحدد ازاي الـ Views تتراص جوا الـ Stack
  • spacing: المسافة بين كل view والتاني

أمثلة عملية:

  • VStack: صفحة تسجيل دخول (عنوان، حقول، أزرار)
  • HStack: شريط التنقل (أيقونة، عنوان، إعدادات)
  • ZStack: صورة مع نص عليها، loading spinner فوق محتوى

نصيحة: ممكن تدمج أكتر من Stack جوا بعض عشان تعمل layouts معقدة!

VStack {
    HStack {
        Text("اليسار")
        Spacer()
        Text("اليمين")
    }
    ZStack {
        Rectangle().fill(Color.gray)
        Text("نص فوق المستطيل")
    }
}
