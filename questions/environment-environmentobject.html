What is Environment and EnvironmentObject in SwiftUI?

Answer in English

Environment:

Environment is a system that allows data to flow automatically down the view hierarchy. It's a way to share values with child views without explicitly passing them as parameters.

// Reading environment values
@Environment(\.colorScheme) var colorScheme
@Environment(\.dismiss) var dismiss

struct ContentView: View {
    @Environment(\.colorScheme) var colorScheme
    
    var body: some View {
        Text("Current scheme: \(colorScheme == .dark ? "Dark" : "Light")")
    }
}

EnvironmentObject:

EnvironmentObject is used to inject observable objects into the environment that can be accessed by any child view in the hierarchy.

class UserSettings: ObservableObject {
    @Published var username = "Guest"
}

struct ParentView: View {
    @StateObject private var settings = UserSettings()
    
    var body: some View {
        ChildView()
            .environmentObject(settings)
    }
}

struct ChildView: View {
    @EnvironmentObject var settings: UserSettings
    
    var body: some View {
        Text("Hello, \(settings.username)!")
    }
}

Key Differences:

Environment EnvironmentObject
For simple values and system values For complex objects that conform to ObservableObject
@Environment(\.keyPath) @EnvironmentObject
Built-in values like colorScheme, dismiss Custom objects you inject

الشرح بالعربي

Environment (البيئة):

دي طريقة في SwiftUI إنك تشارك البيانات مع كل الـ Views اللي تحت view معين من غير ما تضطر تبعت البيانات دي كـ parameters لكل view لوحدها.

مثال: لو عايز تعرف إذا كان التطبيق في Dark Mode ولا Light Mode، هتستخدم:

@Environment(\.colorScheme) var colorScheme

EnvironmentObject (كائن البيئة):

دا بقى للـ Objects الكبيرة اللي عايزها تكون متاحة لكل الـ Views في التطبيق، زي User Settings أو Shopping Cart.

الفرق الأساسي:

  • Environment: للقيم البسيطة وقيم النظام
  • EnvironmentObject: للكائنات المعقدة اللي بتتغير

متى تستخدم إيه:

  • Environment: لما تحتاج قيم النظام زي اللون أو حجم الخط
  • EnvironmentObject: لما عندك object كبير عايز كل الـ views تشوفه
